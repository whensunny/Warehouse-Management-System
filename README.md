# vue_list
```
cd vue_list
```

## 项目环境的安装
```
npm install
```

### 前端VUE的开发的编译和热重载
```
npm run serve //在vue_list目录中
```

### 为生产环境进行编译和微缩
```
npm run build
```

### 自定义配置
See [Configuration Reference](https://cli.vuejs.org/config/).
